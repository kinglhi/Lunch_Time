# Lunch_Time
Simple Android App to order lunch based on a selection of options.
Used Kotlin in Android Studio.
Main focus of project:
  -> DataBinding
  -> LiveData
  -> Fragments
  -> Navigation Graphs
  -> Instrument Testing
  -> Navigation Testing
